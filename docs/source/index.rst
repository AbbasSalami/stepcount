:code:`stepcount`
==================================

A Python package to estimate step counts from accelerometer data.

The algorithm is tuned for wrist-worn AX3 data collected at 100 Hz, making it
compatible with the 
`UK Biobank Accelerometer Dataset <https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0169649>`_.
